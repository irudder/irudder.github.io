生成自签名证书

完整的ssl证书分为四个部分：

  • CA根证书（root CA）
  • 中级证书（Intermediate Certificate）
  • 域名证书
  • 证书秘钥（仅由开发者提供）

证书相当于公钥，pem相当于私钥。

Self-Signed Certificates：包含公钥和私钥的结合体，证书（公钥）会在连接请求的时候发给浏览器，以便浏览器解密和加密。
